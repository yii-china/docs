# 缓存

缓存是提高应用程序性能的一种廉价而有效的方法。
通过将相对静态的数据存储在缓存中,并在请求时从缓存中提供,
应用程序节省了每次从头生成数据所需的时间。

缓存可以在应用程序的不同级别和位置发生。在服务器端,在较低级别,
缓存可用于存储基本数据,例如从数据库获取的最新文章信息列表;
在较高级别,缓存可用于存储网页的片段或整个网页,例如最新文章的渲染结果。
在客户端,您可以使用 HTTP 缓存将最近访问的页面内容保存在浏览器缓存中。

Yii 支持所有这些缓存机制:

* [数据缓存](data.md)
* [片段缓存](fragment.md)
* [页面缓存](page.md)
* [HTTP 缓存](http.md)
