# 您需要了解什么？

Yii 的学习曲线不像其他 PHP 框架那样陡峭，但在开始使用 Yii 之前，仍有一些事情您应该学习。

## PHP

Yii 是一个 PHP 框架，因此请确保您[阅读并理解语言参考](https://www.php.net/manual/zh/langref.php)。

## 面向对象编程

您需要对面向对象编程有基本的了解。如果您不熟悉它，请查看许多可用的教程之一，例如 [tuts+ 的教程](https://code.tutsplus.com/tutorials/object-oriented-php-for-beginners--net-12762)。

当您使用 Yii 进行开发时，您以面向对象的方式编写代码，因此请确保您熟悉 [PHP OOP 支持](https://www.php.net/manual/zh/language.oop5.php)。

请注意，您的应用程序越复杂，您应该学习更高级的 OOP 概念以成功管理该复杂性。

## 命令行和 Composer

Yii 广泛使用事实上的标准 PHP 包管理器 [Composer](https://getcomposer.org)，因此请确保您阅读并理解其[指南](https://getcomposer.org/doc/01-basic-usage.md)。如果您不熟悉使用命令行，现在是开始尝试的时候了。一旦您学习了基础知识，您将永远不想在没有它的情况下工作。

## HTTP

由于 Yii 是一个 Web 框架，而 Web 主要使用 HTTP，因此[了解更多关于它的信息](https://developer.mozilla.org/zh-CN/docs/Web/HTTP)是个好主意。

## Docker

默认应用程序模板利用 Docker，
因此我们建议您[阅读并理解这些概念](https://docs.docker.com/get-started/)。

此外，您将受益于熟悉 [twelve-factor app](https://12factor.net/) 原则。
