# Yii3 实用手册

Yii3 实用手册是一本免费的书籍，包含了关于 PHP 框架 [Yii3](https://www.yiiframework.com/) 的技巧和秘诀。

- 本书由 Yii 社区创建。
- 由 Yii 团队维护和编辑。

欢迎提交 pull request 贡献你的实用技巧。我们会进行审查并将你的技巧添加到本书中。

本书遵循 [Yii 文档规则](https://www.yiiframework.com/license#docs)。
